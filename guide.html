Home Services Application - GUIDE

Step through the Home Services process (in DEV):

  • Create a new Home Services project

    • Upon completion, you should receive a "project created" email

  • Go to the Admin Tool to view details of the project you created

    • Log in using any username and password = "test"

  • From the Admin Tool, click "Approve" or "Decline"

    • You should see the project status change, and you should get an appropriate email in the project creator's inboxOK

  • From the "approved" email (or from here) you can click to the "Congratulations" page
  • From the "Congratulations" page, click "buy" to go to the satisfaction guarantee buy flow
  • Click through to "Checkout". Then you can click "Pay with PayPal"

    • You should see the Payal checkout screen, with the Satisfaction Guarantee line item.

    Note: you can log in to our test PayPal account using "fractalien11@gmail.com/Success11"

  • You should then be brought to the "You're Covered" page

    • You should also receive a confirmation email in the project creator's inbox

NOTE: You will only receive an email (in DEV) if you use an @trustcloud.com email address.
